# jeebase-portal

#### 介绍
jeebase-portal是前端页面。Jeebase是一款前后端分离的开源开发框架，基于springboot+vue（vue-element-admin）开发，二期会整合react前端框架Ant Design React。在实际应用中已经使用这套框架开发了CMS网站系统，社区论坛系统，微信小程序，微信服务号等，后面会逐步整理开源。 本项目主要目的在于整合主流技术框架，寻找应用最佳项目实践方案，实现可直接使用的快速开发框架。
